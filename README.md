# Доставка посылки

Мини-приложение для оформления заявок на доставку посылок с трёхшаговой формой и историей заявок.

## Описание

- **Форма оформления** — 3 шага: данные отправителя, получателя и посылки, подтверждение
- **История заявок** (`/orders`) — список, поиск по имени и городу, фильтр по типу груза
- **Страница заявки** (`/orders/[id]`) — полные детали по клику
- Данные хранятся в `localStorage`

## Стек

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Zod** — валидация форм
- Без UI-библиотек

## Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

```bash
# Сборка
npm run build

# Запуск production-сборки
npm start
```
